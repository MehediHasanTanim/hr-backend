# PDFMake Fonts

Place the following Roboto font files in this directory for payslip PDF generation:

- `Roboto-Regular.ttf`
- `Roboto-Medium.ttf`
- `Roboto-Italic.ttf`

You can download these from Google Fonts:
https://fonts.google.com/specimen/Roboto

Alternatively, copy from the system:
- macOS: `/System/Library/Fonts/Supplemental/` (Helvetica, Arial)
- Or install via: `npm install @pdfmake/fonts`

## Font Detection

The processor will log a warning if fonts are missing but will attempt to continue.
